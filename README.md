# Stochastic-Physics-Sim
**Computational Physics Project: Monte Carlo Decay Modeling**

### Research Objective
To simulate the stochastic nature of radioactive decay and validate the convergence of random sampling toward theoretical exponential decay laws.

### Methodology
- **Sampling:** Monte Carlo algorithm using NumPy `random` distributions.
- **Analysis:** Real-time population tracking and Half-life ($t_{1/2}$) calculation.
- **Environment:** Ubuntu Linux, Python 3.11.
